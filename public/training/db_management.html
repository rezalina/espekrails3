Database Management


Creating database using shell command prompt

Command to create database: createdb
Command to delete database: dropdb

For instance,
to create database mydatabase

>createdb mydatabase
CREATE DATABASE

To delete, simply type

>dropdb mydatabase
DROP DATABASE

Creating database using psql client

To create database mydatabase

#CREATE DATABASE mydatabase
CREATE DATABASE

You are done. Very simple.

Then you can start building tables in your database.
