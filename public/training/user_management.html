User Management


Creating user using shell command prompt

Command to create user: createuser

At the command prompt,try typing

createuser --help

the output will be as below:
createuser creates a new PostgreSQL user.

Usage:
createuser [OPTION]... [USERNAME]

Options:
-a, --adduser user can add new users
-A, --no-adduser user cannot add new users
-d, --createdb user can create new databases
-D, --no-createdb user cannot create databases
-P, --pwprompt assign a password to new user
-E, --encrypted encrypt stored password
-N, --unencrypted do not encrypt stored password
-i, --sysid=SYSID select sysid for new user
-e, --echo show the commands being sent to the server
-q, --quiet don't write any messages
--help show this help, then exit
--version output version information, then exit

Connection options:
-h, --host=HOSTNAME database server host or socket directory
-p, --port=PORT database server port
-U, --username=USERNAME user name to connect as (not the one to create)
-W, --password prompt for password to connect

Example
To create user ahmad with password mypassword

>createuser -P ahmad
Enter password for new user:******
Enter it again:******
Shall the new user be allowed to create databases? (y/n) n
Shall the new user be allowed to create more new users? (y/n) n
Password:
CREATE USER

Creating user using psql client

To create user ahmad with password mypassword

#CREATE USER ahmad PASSWORD 'mypassword'

press enter, and you are done.
